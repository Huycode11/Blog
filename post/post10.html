Hàm (Function) trong JavaScript

Trong JavaScript, hàm (function) là một trong những khối xây dựng quan trọng nhất. Hàm giúp bạn nhóm các đoạn mã thực hiện một nhiệm vụ cụ thể, giúp mã nguồn ngắn gọn, dễ đọc và dễ bảo trì hơn.

1. Khái niệm về hàm

Một hàm trong JavaScript là một tập hợp các câu lệnh được định nghĩa một lần và có thể được gọi lại nhiều lần ở bất cứ đâu trong chương trình.


// Cú pháp cơ bản
function tenHam(thamSo1, thamSo2) {
  // Thân hàm
  return thamSo1 + thamSo2;
}

// Gọi hàm
let tong = tenHam(5, 3);
console.log(tong); // Kết quả: 8

Hàm có thể có tham số hoặc không, có thể trả về giá trị hoặc không tuỳ theo nhu cầu sử dụng.

2. Các loại hàm trong JavaScript

2.1. Hàm khai báo (Function Declaration)

Đây là loại hàm phổ biến nhất, được định nghĩa bằng từ khóa function và có thể được gọi ở bất cứ đâu (hoisting).


function sayHello() {
  console.log("Xin chào, tôi là Văn Huy!");
}

sayHello(); // Gọi hàm

2.2. Hàm biểu thức (Function Expression)

Hàm này được gán cho một biến, không thể được gọi trước khi khai báo.


const chao = function() {
  console.log("Xin chào từ function expression!");
};

chao();

2.3. Hàm mũi tên (Arrow Function)

Hàm mũi tên được giới thiệu trong ES6, cú pháp ngắn gọn và không có this riêng.


const tong = (a, b) => a + b;
console.log(tong(5, 10)); // 15

Lưu ý: Arrow function không thể dùng làm constructor và không có arguments riêng.

3. Hàm có tham số mặc định

Từ ES6, bạn có thể đặt giá trị mặc định cho tham số, giúp code linh hoạt và tránh lỗi thiếu tham số.


function greet(name = "Khách") {
  console.log("Xin chào, " + name + "!");
}

greet("Huy"); // Xin chào, Huy!
greet(); // Xin chào, Khách!

4. Hàm ẩn danh và callback

Hàm ẩn danh (anonymous function) không có tên, thường được dùng làm callback trong các tình huống như sự kiện hoặc xử lý bất đồng bộ.


setTimeout(function() {
  console.log("Hàm này chạy sau 2 giây");
}, 2000);

Ví dụ callback trong mảng


let numbers = [1, 2, 3];
numbers.forEach(function(num) {
  console.log("Giá trị:", num);
});

5. Hàm lồng nhau và phạm vi (Scope)

Bạn có thể khai báo một hàm bên trong hàm khác. Hàm con chỉ có thể truy cập được trong phạm vi của hàm cha.


function outer() {
  let outerVar = "Biến bên ngoài";

  function inner() {
    console.log(outerVar);
  }

  inner(); // Có thể truy cập outerVar
}

outer();

Hiểu rõ phạm vi giúp bạn tránh lỗi biến bị ghi đè hoặc không xác định.

6. Hàm đệ quy (Recursive Function)

Hàm có thể tự gọi lại chính nó để giải quyết bài toán lặp có cấu trúc, như tính giai thừa hoặc duyệt cây.


function factorial(n) {
  if (n === 0) return 1;
  return n * factorial(n - 1);
}

console.log(factorial(5)); // Kết quả: 120

7. Tổng kết

  • Hàm giúp tổ chức mã nguồn rõ ràng và dễ tái sử dụng.
  • Arrow function thích hợp cho các tác vụ ngắn gọn, không cần this.
  • Callback và đệ quy là hai khái niệm nâng cao quan trọng cần nắm vững.
