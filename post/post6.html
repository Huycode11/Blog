Java File I/O – Đọc và ghi tệp trong Java

Tác giả: Văn Huy | Ngày đăng: 21/10/2025

1. Giới thiệu

Trong lập trình Java, **File I/O (Input/Output)** cho phép chúng ta thao tác với tệp — bao gồm đọc dữ liệu từ tệp, ghi dữ liệu vào tệp, và xử lý lỗi trong quá trình truy cập tệp. Đây là một phần quan trọng giúp chương trình có thể lưu trữ dữ liệu bền vững.

2. Các lớp quan trọng trong Java I/O

Java cung cấp nhiều lớp để xử lý tệp trong gói java.io:

  • File – đại diện cho đường dẫn tệp hoặc thư mục.
  • FileReader, FileWriter – đọc/ghi ký tự.
  • BufferedReader, BufferedWriter – đọc/ghi hiệu quả hơn.
  • FileInputStream, FileOutputStream – đọc/ghi byte (nhị phân).

3. Tạo và kiểm tra tệp


import java.io.File;
import java.io.IOException;

public class CreateFileExample {
    public static void main(String[] args) {
        File file = new File("example.txt");
        try {
            if (file.createNewFile()) {
                System.out.println("Tệp đã được tạo: " + file.getName());
            } else {
                System.out.println("Tệp đã tồn tại.");
            }
        } catch (IOException e) {
            System.out.println("Lỗi khi tạo tệp.");
            e.printStackTrace();
        }
    }
}

4. Ghi dữ liệu vào tệp


import java.io.FileWriter;
import java.io.IOException;

public class WriteFileExample {
    public static void main(String[] args) {
        try {
            FileWriter writer = new FileWriter("example.txt");
            writer.write("Xin chào, đây là nội dung được ghi bằng Java!");
            writer.close();
            System.out.println("Ghi dữ liệu thành công.");
        } catch (IOException e) {
            System.out.println("Đã xảy ra lỗi khi ghi tệp.");
            e.printStackTrace();
        }
    }
}

5. Đọc dữ liệu từ tệp


import java.io.File;
import java.io.FileNotFoundException;
import java.util.Scanner;

public class ReadFileExample {
    public static void main(String[] args) {
        try {
            File file = new File("example.txt");
            Scanner reader = new Scanner(file);
            while (reader.hasNextLine()) {
                String data = reader.nextLine();
                System.out.println(data);
            }
            reader.close();
        } catch (FileNotFoundException e) {
            System.out.println("Không tìm thấy tệp.");
            e.printStackTrace();
        }
    }
}

6. Đọc/Ghi nhị phân (Binary Stream)


import java.io.FileOutputStream;
import java.io.FileInputStream;
import java.io.IOException;

public class BinaryExample {
    public static void main(String[] args) throws IOException {
        // Ghi tệp
        FileOutputStream out = new FileOutputStream("data.bin");
        out.write(100);
        out.write(200);
        out.close();

        // Đọc tệp
        FileInputStream in = new FileInputStream("data.bin");
        int value;
        while ((value = in.read()) != -1) {
            System.out.println(value);
        }
        in.close();
    }
}

7. Xử lý lỗi trong I/O

Khi thao tác với tệp, ta nên sử dụng **try-catch-finally** hoặc **try-with-resources** để đảm bảo tệp được đóng đúng cách, tránh rò rỉ tài nguyên. 


try (FileWriter writer = new FileWriter("example.txt")) {
    writer.write("Ghi dữ liệu an toàn hơn với try-with-resources!");
} catch (IOException e) {
    e.printStackTrace();
}

9. Kết luận

Kiến thức về File I/O là nền tảng quan trọng trong Java, giúp bạn xây dựng các ứng dụng lưu trữ và xử lý dữ liệu thực tế. Việc hiểu rõ cách đọc, ghi và xử lý lỗi sẽ giúp chương trình ổn định, an toàn và hiệu quả hơn.
