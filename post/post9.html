Cấu trúc điều kiện và vòng lặp trong JavaScript

Trong quá trình lập trình, việc ra quyết định và thực hiện các thao tác lặp lại là hai khái niệm vô cùng quan trọng. JavaScript cung cấp các cấu trúc điều kiện và vòng lặp giúp lập trình viên xử lý linh hoạt luồng chương trình.

1. Cấu trúc điều kiện trong JavaScript

Các cấu trúc điều kiện cho phép chương trình thực hiện các đoạn mã khác nhau dựa trên giá trị của biểu thức logic.

1.1. Lệnh if


let age = 20;
if (age >= 18) {
  console.log("Bạn đã đủ tuổi trưởng thành");
}

1.2. Lệnh if...else


let score = 75;
if (score >= 50) {
  console.log("Bạn đã đậu");
} else {
  console.log("Bạn đã rớt");
}

1.3. Lệnh if...else if...else


let grade = 85;

if (grade >= 90) {
  console.log("Xuất sắc");
} else if (grade >= 70) {
  console.log("Khá");
} else {
  console.log("Trung bình");
}

1.4. Cấu trúc switch

Cấu trúc switch giúp so sánh một giá trị với nhiều trường hợp khác nhau, thường được dùng khi có nhiều nhánh lựa chọn.


let day = 3;
switch(day) {
  case 1:
    console.log("Thứ Hai");
    break;
  case 2:
    console.log("Thứ Ba");
    break;
  case 3:
    console.log("Thứ Tư");
    break;
  default:
    console.log("Ngày không hợp lệ");
}

2. Vòng lặp trong JavaScript

Vòng lặp cho phép bạn thực hiện lặp đi lặp lại một đoạn mã khi điều kiện còn đúng. JavaScript hỗ trợ nhiều loại vòng lặp phổ biến.

2.1. Vòng lặp for


for (let i = 1; i <= 5; i++) {
  console.log("Lần lặp thứ " + i);
}

2.2. Vòng lặp while


let count = 0;
while (count < 3) {
  console.log("Giá trị hiện tại: " + count);
  count++;
}

2.3. Vòng lặp do...while

Khác với while, vòng lặp do...while luôn thực hiện ít nhất một lần trước khi kiểm tra điều kiện.


let num = 0;
do {
  console.log("Giá trị: " + num);
  num++;
} while (num < 3);

2.4. Vòng lặp for...of và for...in

  • for...of dùng để lặp qua các phần tử của mảng.
  • for...in dùng để lặp qua các thuộc tính của đối tượng.

let fruits = ["Táo", "Cam", "Chuối"];
for (let fruit of fruits) {
  console.log(fruit);
}

let person = { name: "Huy", age: 21, city: "TP.HCM" };
for (let key in person) {
  console.log(key + ": " + person[key]);
}

3. Tối ưu hoá và lưu ý

  • Tránh sử dụng vòng lặp vô hạn nếu không có điều kiện dừng.
  • Sử dụng break và continue để điều khiển luồng vòng lặp hợp lý.
  • Khi cần xử lý dữ liệu mảng, cân nhắc dùng các phương thức như forEach, map, filter để ngắn gọn hơn.
