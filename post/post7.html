JavaScript Cơ Bản: Làm Quen Và Ứng Dụng

Đăng ngày: 20/10/2025

JavaScript là ngôn ngữ lập trình của Web. Nó cho phép bạn thêm logic, chuyển động và tương tác vào các trang HTML tĩnh. Nếu HTML là phần “xương sống”, CSS là “vẻ ngoài”, thì JavaScript chính là “trái tim” khiến trang web sống động.

🔹 1. JavaScript Là Gì?

JavaScript (JS) là ngôn ngữ lập trình chạy trên trình duyệt, giúp tương tác với người dùng và điều khiển nội dung trang web mà không cần tải lại trang.

  • HTML: Cấu trúc nội dung.
  • CSS: Định dạng và tạo phong cách.
  • JavaScript: Tương tác, xử lý dữ liệu, logic.
“JavaScript giúp biến trang web từ tĩnh sang động — người dùng có thể click, nhập dữ liệu, hoặc xem thay đổi ngay lập tức.”

🔹 2. Cách Thêm JavaScript Vào Trang

Có 3 cách cơ bản để thêm JavaScript vào trang web:

<!-- 1. Nội tuyến -->
<button onclick="alert('Xin chào!')">Nhấn tôi</button>

<!-- 2. Trong thẻ script -->
<script>
  alert("Chào mừng đến với JS!");
</script>

<!-- 3. Tách riêng file .js -->
<script src="main.js"></script>

🔹 3. Cấu Trúc Cơ Bản

Giống như các ngôn ngữ khác, JavaScript có biến, toán tử, hàm, điều kiện và vòng lặp. Dưới đây là ví dụ đơn giản:

// Khai báo biến
let name = "Văn Huy";
let age = 21;

// Xuất ra console
console.log("Xin chào, tôi là " + name + ", " + age + " tuổi!");

// Câu điều kiện
if (age >= 18) {
  console.log("Bạn đã đủ tuổi trưởng thành!");
}

🔹 4. Làm Việc Với DOM

DOM (Document Object Model) cho phép bạn truy cập và thay đổi nội dung HTML.

<p id="demo">Xin chào</p>
<button onclick="changeText()">Thay đổi nội dung</button>

<script>
  function changeText() {
    document.getElementById("demo").innerHTML = "Bạn vừa nhấn nút!";
  }
</script>
💡 Mẹo: Sử dụng document.querySelector() để chọn phần tử nhanh và linh hoạt hơn.

🔹 5. Xử Lý Sự Kiện

JavaScript hỗ trợ rất nhiều loại sự kiện như: click, mouseover, keydown, submit,...

<input type="text" id="nameInput" placeholder="Nhập tên của bạn...">
<button id="btnShow">Chào</button>

<script>
  document.getElementById("btnShow").addEventListener("click", function() {
    let name = document.getElementById("nameInput").value;
    alert("Xin chào, " + name + "!");
  });
</script>

🔹 6. Tương Tác Với CSS

Bạn có thể thay đổi kiểu CSS bằng JS để tạo hiệu ứng động:

<div id="box" style="width:100px;height:100px;background:red;"></div>
<button onclick="changeColor()">Đổi màu</button>

<script>
  function changeColor() {
    document.getElementById("box").style.backgroundColor = "blue";
  }
</script>

🔹 7. Kết Luận

JavaScript là nền tảng để học các công nghệ cao hơn như React, Node.js, và Vue. Việc nắm vững các khái niệm cơ bản như biến, hàm, và DOM sẽ giúp bạn dễ dàng tiến xa trong lập trình web hiện đại.

👉 Trong bài tiếp theo, chúng ta sẽ tìm hiểu sâu về biến và kiểu dữ liệu trong JavaScript.
