Java Basics: Getting Started

Ngày đăng: Oct 17, 2025

Java là một trong những ngôn ngữ lập trình phổ biến nhất trên thế giới. Nó được thiết kế với triết lý “Write Once, Run Anywhere” — tức là bạn chỉ cần viết chương trình một lần và có thể chạy trên bất kỳ nền tảng nào có cài đặt Java Virtual Machine (JVM).

1️⃣ Cài đặt môi trường

Trước khi bắt đầu, bạn cần cài đặt:

  • JDK (Java Development Kit) — để biên dịch và chạy chương trình Java.
  • Một IDE như VS Code, IntelliJ IDEA hoặc Eclipse.

2️⃣ Viết chương trình đầu tiên

Sau khi cài đặt xong, hãy tạo một file tên HelloWorld.java và nhập đoạn mã sau:

public class HelloWorld {
  public static void main(String[] args) {
    System.out.println("Hello, World!");
  }
}

Khi chạy chương trình, bạn sẽ thấy kết quả:

Hello, World!

3️⃣ Giải thích mã

  • public class HelloWorld: Định nghĩa một lớp tên là HelloWorld.
  • public static void main(String[] args): Là điểm bắt đầu của chương trình Java.
  • System.out.println(): Dùng để in ra màn hình.

4️⃣ Kết luận

Qua bài viết này, bạn đã biết cách cài đặt môi trường và viết chương trình Java đầu tiên. Bước tiếp theo, bạn có thể tìm hiểu về biến, kiểu dữ liệu, và các cấu trúc điều khiển trong Java để phát triển kỹ năng của mình hơn nữa.
